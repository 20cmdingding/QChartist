high1(bar_number as integer) as double

contains the highest price of the specified bar number for the chart that is currently displayed.
Bar number : 0 is the first bar (with the most recent date time), from the right to the left.

These are high prices for the 1 minute chart period (timeframe).
This array is filled when you attribute a specified timeframe to the current chart. There are also arrays (high5(), high15(), ...) for all the other timeframes. Useful for multi timeframe indicators, chart overlays...
