time1(bar_number as integer) as string

contains the time (HH:MM) of the specified bar number for the chart that is currently displayed.
Bar number : 0 is the first bar (with the most recent date time), from the right to the left.

These are time for the 1 minute chart period (timeframe).
This array is filled when you attribute a specified timeframe to the current chart. There are also arrays (time5(), time15(), ...) for all the other timeframes. Useful for multi timeframe indicators, chart overlays...
