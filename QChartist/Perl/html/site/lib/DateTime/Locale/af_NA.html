NAME

DateTime::Locale::af_NA

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'af_NA' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Afrikaans Namibia.

This locale inherits from the DateTime::Locale::af locale.
