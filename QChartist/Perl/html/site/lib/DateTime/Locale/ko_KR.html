NAME

DateTime::Locale::ko_KR

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ko_KR' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Korean South Korea.

This locale inherits from the DateTime::Locale::ko locale.
