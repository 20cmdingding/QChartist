NAME

DateTime::Locale::mt_MT

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'mt_MT' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Maltese Malta.

DATA

This locale inherits from the DateTime::Locale::mt locale.
