NAME

DateTime::Locale::he

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'he' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Hebrew.

DATA

This locale inherits from the DateTime::Locale::root locale.
