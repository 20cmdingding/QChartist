NAME

DateTime::Locale::pa_Guru

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'pa_Guru' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Punjabi Gurmukhi.
