NAME

DateTime::Locale::bg_BG

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'bg_BG' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Bulgarian Bulgaria.

DATA

This locale inherits from the DateTime::Locale::bg locale.
