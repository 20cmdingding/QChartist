NAME

DateTime::Locale::ru_UA

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ru_UA' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Russian Ukraine.

DATA

This locale inherits from the DateTime::Locale::ru locale.
