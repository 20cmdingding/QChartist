NAME

DateTime::Locale::ml_IN

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ml_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Malayalam India.

DATA

This locale inherits from the DateTime::Locale::ml locale.

It contains the following data.
