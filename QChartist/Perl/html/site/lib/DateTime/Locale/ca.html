NAME

DateTime::Locale::ca

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ca' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Catalan.

This locale inherits from the DateTime::Locale::root locale.
