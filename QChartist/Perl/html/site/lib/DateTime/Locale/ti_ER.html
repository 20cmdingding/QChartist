NAME

DateTime::Locale::ti_ER

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ti_ER' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Tigrinya Eritrea.
