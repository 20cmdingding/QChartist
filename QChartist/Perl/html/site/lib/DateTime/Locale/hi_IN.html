NAME

DateTime::Locale::hi_IN

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'hi_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Hindi India.

DATA

This locale inherits from the DateTime::Locale::hi locale.
