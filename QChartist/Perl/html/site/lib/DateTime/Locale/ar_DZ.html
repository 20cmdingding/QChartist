NAME

DateTime::Locale::ar_DZ

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ar_DZ' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Arabic Algeria.

DATA

This locale inherits from the DateTime::Locale::ar locale.

It contains the following data.
