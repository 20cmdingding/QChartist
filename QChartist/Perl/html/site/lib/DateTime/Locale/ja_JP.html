NAME

DateTime::Locale::ja_JP

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ja_JP' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Japanese Japan.
