NAME

DateTime::Locale::be_BY

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'be_BY' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Belarusian Belarus.
