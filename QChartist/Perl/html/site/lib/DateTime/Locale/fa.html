NAME

DateTime::Locale::fa

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'fa' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Persian.

DATA

This locale inherits from the DateTime::Locale::root locale.
