NAME

DateTime::Locale::kk_Cyrl

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'kk_Cyrl' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Kazakh Cyrillic.

This locale inherits from the DateTime::Locale::kk locale.
