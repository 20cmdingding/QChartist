NAME

DateTime::Locale::sq

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'sq' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Albanian.

DATA

This locale inherits from the DateTime::Locale::root locale.
