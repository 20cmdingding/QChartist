NAME

DateTime::Locale::gu_IN

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'gu_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Gujarati India.

DATA

This locale inherits from the DateTime::Locale::gu locale.

It contains the following data.
