NAME

DateTime::Locale::en_NA

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'en_NA' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for English Namibia.
