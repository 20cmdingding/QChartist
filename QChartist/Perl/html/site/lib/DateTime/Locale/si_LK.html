NAME

DateTime::Locale::si_LK

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'si_LK' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Sinhala Sri Lanka.
