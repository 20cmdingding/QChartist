NAME

DateTime::Locale::my_MM

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'my_MM' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Burmese Myanmar [Burma].

This locale inherits from the DateTime::Locale::my locale.
