NAME

DateTime::Locale::ln

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ln' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Lingala.

DATA

This locale inherits from the DateTime::Locale::root locale.
