NAME

DateTime::Locale::ta_IN

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'ta_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Tamil India.

DATA

This locale inherits from the DateTime::Locale::ta locale.

It contains the following data.
