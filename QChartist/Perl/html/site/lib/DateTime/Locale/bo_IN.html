NAME

DateTime::Locale::bo_IN

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'bo_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Tibetan India.

DATA

This locale inherits from the DateTime::Locale::bo locale.

It contains the following data.
