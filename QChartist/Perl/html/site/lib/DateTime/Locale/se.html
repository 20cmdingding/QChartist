NAME

DateTime::Locale::se

SYNOPSIS

  use DateTime;
  
  my $dt = DateTime->now( locale => 'se' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Northern Sami.

DATA

This locale inherits from the DateTime::Locale::root locale.
