NAME

Finance::Quote::Tiaacref - Obtain quote from TIAA-CREF.

SYNOPSIS

    use Finance::Quote;
    
    $q = Finance::Quote->new;
    
    %stockinfo = $q->fetch("tiaacref","TIAAreal");

DESCRIPTION

This module is loaded by default on a Finance::Quote object. It's also possible to load it explicitly by passing "Tiaacref" in to the argument argument list of Finance::Quote->new().

Information returned by this module is governed by TIAA-CREF's terms and conditions.

LABELS RETURNED
