Installation :

For proper functionning of the program, install QChartist into C:\QChartist\
Copy C:\QChartist\Windows\RapidQ32.dll to C:\Windows\
Run QChartist.exe to start the program
Run QChartist.bat to recompile the program if you modify the sources. Don't forget to share your code on the forum http://www.qchartist.net

see docs\readme.txt and docs\QChartist.chm for more info
